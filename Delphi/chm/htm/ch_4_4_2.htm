4.4.2.  Работа со списком таможенных расписок

Чтобы просмотреть список загруженных ТР выполните команду Документы -> ТР на панели навигации.

 

Список ТР

Документы выделяются в списке различными цветами:

Голубым – ТР с ошибками;

Красным – ТР с задолженностью;

Серым  – аннулированные.

 

Чтобы найти ТР:

1.     Нажмите на кнопку  на панели инструментов Списка ТР.

На экране появится панель для задания параметров поиска Поиск объектов.

2.     Задайте критерии поиска, заполнив следующие поля:

    • Тип. номер;
    • Код таможни;
    • Дата с;
    • Дата по;
    • Режим;
    • Метод перем.

 

Примечание

Задание условия производится посредством ввода в поле значения, которому оно должно соответствовать. Для всех полей, кроме дат, можно вводить неполное значение (только начальные символы). Если значения указаны для нескольких полей, то система будет выбирать объекты, удовлетворяющие всем условиям одновременно. В условии можно использовать символы подстановки: “%” – любое количество любых символов, “_” – один произвольный символ.

Например, чтобы найти все объекты, содержащие часть слова пошлина, в параметрах поиска следует указать пошлин% или %ошлин%

Чтобы найти объект, содержащий в номере символ «4», в параметрах поиска следует указать %4% (будут выведены все объекты, содержащие символ «4» в середине номера) или 4% (будут выведены все объекты, содержащие в начале символ «4»).

 

3.     Нажмите на кнопку Искать.

На экран будут выведены результаты поиска, осуществленного в соответствии с заданными параметрами.

 

 

Чтобы просмотреть данные по ТР:

1.     Выберите в списке ТР.

2.     Нажмите на кнопку  Открыть выбранные файлы на панели инструментов окна Список ТР.

На экране появится детальная форма ТР.

На закладке ТР представлены основные данные по таможенной расписке.

 

Закладка ТР детальной формы ТР

На закладке К уплате детальной формы ТР представлен список всех платежей, которые подлежат уплате по таможенной расписке.

 

Закладка К уплате детальной формы ТР

 

На закладке Оплачено детальной формы ТР представлен список всех платежных документов, по которым была произведена оплата ТР.

 

При двойном нажатии левой кнопкой мыши на платежном документе из списка откроется его детальная форма.

 

На закладке Товары представлен список всех товаров и начисления по ним.

 

На закладке Ошибки представлен перечень ошибок, которые возникли в процессе обработки ТПО. 

 

 

Чтобы аннулировать ТР:

  1. Выберите в списке ТР, которую необходимо аннулировать.
  2. Откройте детальную форму ТР.
  3. Нажмите на кнопку Аннулировать на панели инструментов закладки ТР детальной формы ТР.

На экране появится запрос на подтверждение аннулирования выбранной ТР.

  1. Нажмите Да, если данную ТР действительно требуется аннулировать.

Выбранная ТР будет аннулирована и в списке выделяться серым цветом..

 

 

Чтобы удалить ТР:

  1. Выберите в списке ТР, которую необходимо удалить.
  2. Нажмите на кнопку  Удалить выделенные объекты на панели инструментов списка ТР.

На экране появится запрос на подтверждение удаления выбранной ТР.

  1. Нажмите Да, если данную ТР действительно требуется удалить.

Выбранная расписка будет удалена из списка ТР.
