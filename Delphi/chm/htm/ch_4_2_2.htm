4.2.2.  Работа со списком ГТД

Чтобы просмотреть список загруженных ГТД выполните команду Документы -> ГТД на панели навигации.

 

Список ГТД

Документы выделяются в списке различными цветами:

Голубым – ГТД с ошибками;

Красным – ГТД с задолженностью;

Зеленым – ГТД крупных плательщиков.

 

В списке документов предусмотрены следующие виды фильтрации:

Показать:

  • Все – в списке отображаются все документы
  • Только хорошие – в списке отображаются только ГТД без ошибок,
  • Только с ошибками - в списке отображаются только ГТД с ошибками.

Код: - для отбора ГТД по указанному посту.

 

Чтобы найти ГТД:

1.     Нажмите на кнопку  на панели инструментов Списка ГТД.

На экране появится панель для задания параметров поиска Поиск объектов.

2.     Задайте критерии поиска, заполнив следующие поля:

    • Код таможни (поста)
    • Дата ГТД (период);
    • Номер;
    • ИНН (декларанта);
    • КПП (декларанта);

 

Примечание

Задание условия производится посредством ввода в поле значения, которому оно должно соответствовать. Для всех полей, кроме дат, можно вводить неполное значение (только начальные символы). Если значения указаны для нескольких полей, то система будет выбирать объекты, удовлетворяющие всем условиям одновременно. В условии можно использовать символы подстановки: “%” – любое количество любых символов, “_” – один произвольный символ.

Например, чтобы найти все объекты, содержащие часть слова пошлина, в параметрах поиска следует указать пошлин% или %ошлин%

Чтобы найти объект, содержащий в номере символ «4», в параметрах поиска следует указать %4% (будут выведены все объекты, содержащие символ «4» в середине номера) или 4% (будут выведены все объекты, содержащие в начале символ «4»).

 

3.     Нажмите на кнопку Искать.

На экран будут выведены результаты поиска, осуществленного в соответствии с заданными параметрами.

 

 

Чтобы просмотреть данные по ГТД:

1.     Выберите в списке нужную ГТД.

2.     Нажмите на кнопку  Открыть выбранные файлы на панели инструментов окна Список ГТД.

На экране появится детальная форма ГТД.

 

На закладке ГТД  представлены основные данные по ГТД.

 

Закладка ГТД детальной формы ГТД

На закладке К уплате детальной формы ГТД представлен список всех платежей, которые должны быть уплачены по декларации.

Закладка К уплате детальной формы ГТД

В этом режиме также доступны следующие возможности:

 - просмотреть историю изменения начислений по выделенному виду;

 - закрыть временную ГТД (подробнее описано в документе «Описание обработки временных и полных деклараций)

 - отменить закрытие ГТД

 

На закладке Оплачено детальной формы ГТД представлен список всех платежных документов, по которым была произведена оплата по данной декларации.

 

Закладка Оплачено детальной формы ГТД

С помощью кнопки  Показать виды платежей можно менять вид списка:

при отжатой (выключенной) кнопке – показываются списания с платежных документов без разбивки по видам таможенных платежей;

при нажатой (включенной) кнопке – показываются списания с платежных документов с разбивкой по видам таможенных платежей и датам списания;

 

При двойном нажатии левой кнопкой мыши на платежном документе из списка откроется его детальная форма.

 

На закладке Товары представлен список всех товаров и начисления по ним.

 

Для просмотра ГТД в полном объеме нажмите на кнопку  Просмотр полного документа на панели инструментов на закладке ГТД.

 

Просмотр полной ГТД – основной лист

Для просмотра дополнительных листов ГТД нажмите на кнопку  на панели инструментов.

 

Просмотр полной ГТД – дополнительный лист

 

В этом режиме также доступны следующие возможности:

 - открыть ранее сохраненный документ

 - сохранить документ

 - распечатать документ

 - перейти к первой странице документа

 - перейти к предыдущей странице документа

 - перейти к следующей странице документа

 - перейти к последней странице документа

 - перейти к указанной в окошке странице документа

 - увеличить документ

 - уменьшить документ

 - задать размер документа по ширине листа

 - задать размер документа лист целиком

 - задать размер документа (в %)

 - выйти из режима просмотра.

 

 

Чтобы создать новую ГТД:

  1. Нажмите на кнопку  Добавить новый объект на панели инструментов Списка ГТД.

На экране появится детальная форма ГТД.

  1. Заполните необходимые поля и нажмите на кнопку  Сохранить изменения.

В детальной форме ГТД появятся закладки.

  1. Перейдите на закладку К уплате.
  2. Нажмите на кнопку  Добавить новый объект на панели инструментов детальной формы ГТД.

На экране появится детальная форма Нового начисления по ГТД.

  1. Заполните необходимые поля и нажмите на кнопку   Сохранить изменения.
  2. Закройте детальную форму Нового начисления по ГТД.

Создайте необходимое количество начислений.

  1. Перейдите на закладку Товары.
  2. Нажмите на кнопку  Добавить новый объект на панели инструментов детальной формы ГТД, чтобы добавить товар в ГТД.

На экране появится детальная форма Товара.

  1.  Заполните необходимые поля и нажмите на кнопку  Сохранить изменения.
  2. Закройте детальную форму Товара.

Создайте необходимое количество товаров в ГТД.

 

 

Чтобы удалить ГТД:

  1. Выберите в списке ГТД, которую необходимо удалить.
  2. Нажмите на кнопку  Удалить выделенные объекты на панели инструментов списка ГТД.

На экране появится запрос на подтверждение удаления выбранной ГТД.

  1. Нажмите Да, если данную ГТД действительно требуется удалить.

Выбранная декларация будет удалена из списка ГТД.

 

 

Примечания

  1. При изменении начислений удалять старую ГТД и загружать новую не корректно – это приводит к изменению отчетности, поскольку в этом случае первоначальные начисления по ГТД удаляются из первоначального дня загрузки, а при повторной загрузке новые начисления полностью отразятся в текущем дне. В этом случае необходимо пользоваться режимом корректировки начислений - в этом случае первоначальные начисления останутся без изменений, а в текущем дне отразятся только суммы корректировки.

 

  1. ГТД не может быть удалена, если:

-       По нему уже отправлены суммы для перечисления в бюджет;

-       Закрыт день, в котором он был введен.
