4.5.       Прочие списания

Данный режим предназначен для списания средств когда нет ни ГТД, ни ТПО.

 

Данные по прочим списаниям доступны в меню панели навигации Документы -> Прочие списания.

 

 

Для выполнения операций с документами по прочим списаниям воспользуйтесь кнопками на панели инструментов.

 

Чтобы найти документ:

  1. Нажмите на кнопку  на панели инструментов списка Прочие списания.

На экране появится панель для задания параметров поиска Поиск объектов.

  1. Задайте критерии поиска, заполнив следующие поля:
    • Наименование;
    • ИНН;
    • КПП;
    • Номер;
    • Дата с;
    • Дата по;

 

Примечание

Задание условия производится посредством ввода в поле значения, которому оно должно соответствовать. Для всех полей, кроме дат, можно вводить неполное значение (только начальные символы). Если значения указаны для нескольких полей, то система будет выбирать объекты, удовлетворяющие всем условиям одновременно. В условии можно использовать символы подстановки: “%” – любое количество любых символов, “_” – один произвольный символ.

Например, чтобы найти все объекты, содержащие часть слова пошлина, в параметрах поиска следует указать пошлин% или %ошлин%

Чтобы найти объект, содержащий в номере символ «4», в параметрах поиска следует указать %4% (будут выведены все объекты, содержащие символ «4» в середине номера) или 4% (будут выведены все объекты, содержащие в начале символ «4»).

 

  1. Нажмите на кнопку Искать.

На экран будут выведены результаты поиска, осуществленного в соответствии с заданным параметром.

 

Чтобы создать новый документ на списание:

  1. Нажмите на кнопку  Добавить новый объект на панели инструментов Списка возвратов.

На экране появится окно Типов создаваемых объектов.

 

Окно «Тип создаваемого объекта»

 

  1. Выберите из списка нужный тип и нажмите на кнопку ОК.

На экране появится детальная форма Нового документа.

 

 

  1.  Сохранить изменения.
  2. Перейдите на закладку Начисления.
  3. Нажмите на кнопку  Добавить новый объект на панели инструментов.
  4. На экране появится окно Новое начисление.

Заполните все поля и нажмите на кнопку Далее.

 

 

  1. Данную сумму можно либо оплатить, либо перевести её в задолженность.

 

 

  1. Выберите необходимое действие.

Если выбрать Перевести в задолженность, нажмите на кнопку Готово.

Если выбрать Искать платежные поручения, нажмите на кнопку Далее.

  1. Появится Мастер оплаты.

 

 

  1. Нажмите на кнопку Добавить.
  2. Появится Список платежных документов. Найдите нужный документ и нажмите на кнопку Выбрать. В появившемся окне введите сумму списания с документа и нажмите ОК.

 

 

  1. Если сумма списания меньше суммы корректировки, то в мастере оплаты можно добавить еще один платежный документ или поставить галочку Перевести остаток в задолженность.
  2. Нажмите на кнопку Готово.
  3. Аналогичным образом можно добавить еще один вид платежа, повторив действия 5-13.
  4. По окончании ввода всех сумм закройте окно.

 

 

Чтобы просмотреть данные документа на списание:

  1. Выберите в списке нужный документ.
  2. Нажмите на кнопку  Открыть выбранные файлы на панели инструментов окна Прочие списания.

На экране появится детальная форма Документа на списание.

 

 

Чтобы удалить документ на списание:

  1. Выберите в списке документ на возврат, который необходимо удалить.
  2. Нажмите на кнопку  Удалить выделенные объекты на панели инструментов списка возвратов.

На экране появится запрос на подтверждение удаления выбранного документа на возврат.

  1. Нажмите Да, если данный документ на возврат действительно требуется удалить.

Выбранный документ будет удалена из списка возвратов.
