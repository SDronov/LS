3.5.1.  Общие сведения

Для первоначального запуска КПС в работу необходимо ввести начальные остатки.

Ввод начальных остатков должен осуществляться на определенную дату.

 

Вводу подлежат следующие данные:

 

№ п/п

Наименование данных

1.            

Остатки на счетах участников ВЭД и невыясненные суммы в разрезе каждого платежного документа (т.е. фактически остатки по платежным документам)

2.            

Платежные документы, еще не поступившие на счета таможни (платежки на контроле) и суммы списаний по ним (деньги в пути)

3.            

Суммы задолженности по уплате таможенных и иных платежей

4.            

Перечень ГТД, находящихся в режиме временного ввоза, со сведениями о количестве оплаченных месяцев на момент ввода начальных остатков

 

Начальные остатки могут быть вводиться различными способами:

-       путем переноса данных из предыдущей версии КПС «Лицевые счета» ИРС «Доход» (версия 2.11);

-       путем ввода из текстовых файлов;

-       ручным вводом данных.

Ввод данных осуществляется через специальный буфер.

 

Для ввода начальных остатков предназначен отдельный модуль «ИРС Доход – Перенос остатков».

 

Чтобы запустить модуль переноса остатков:

1.     Воспользуйтесь ярлыком запуска «ИРС Доход – Перенос остатков», созданном при установке программы в меню Пуск/Программы/ИРС Доход/Лицевые счета.

 

Примечание

Если ярлык запуска «ИРС Доход – Перенос остатков» отсутствует в папке «Лицевые счета», значит модуль не был установлен. Подробнее об установке данного модуля описано в Руководство по начальной установке системы.

 

На экране появится диалоговое окно входа в систему.

 

 

2.     Для входа в модуль введите имя пользователя, пароль и имя базы данных, такие же как и при входе в основную программу. Программа запоминает последние успешно введенные данные и при следующем входе программы достаточно будет ввести только пароль.

3.     Нажмите на кнопку ОК.

 

После входа в программу появится главное окно модуля.

 

 

В нем доступны следующие возможности:

Загрузить… - загрузить данные по остаткам из КПС «Лицевые счета» версии 2.11 или из текстовых файлов;

Очистить – очистить все данные в буфере;

Перенести – перенести данные из буфера в основную базу;

Выход – выйти из модуля.

 

ВНИМАНИЕ!

При очистке буфера также удаляются все перенесенные из него данные в основной базе. Поэтому после переноса данных буфер очищать не надо.

 

 

На панели навигации доступно 2 режима:

Список ГТД – работа со списком ГТД в буфере;

Список плат. док. – работа со списком платежных документов в буфере.
