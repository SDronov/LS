4.7.       Перезачеты

Данный режим предназначен для формирования перезачетов средств плательщикам.

Перезачет может формироваться двумя способами:

  1. При закрытии временной декларации (подробнее об этом в документе «Описание обработки временных и полных деклараций»).
  2. Когда остатки по платежным документам надо перезачесть на новый документ - Письмо по остатку.

 

Данные по перезачетам доступны в меню панели навигации Документы -> Перезачеты.

 

 

Для выполнения операций с документами на перезачет воспользуйтесь кнопками на панели инструментов.

 

Чтобы создать новый документ на перезачет:

  1. Нажмите на кнопку  Добавить новый объект на панели инструментов Списка перезачетов.

На экране появится детальная форма Заявления на перезачет.

 

 

  1. Заполните необходимые поля.

В поле Плательщик нажмите на кнопку . На экране появится окно Список участников ВЭД. Выберите из списка нужного участника и нажмите на кнопку Выбрать.

  1. Нажмите на кнопку  Сохранить изменения.
  2. Перейдите на закладку Связанные документы.
  3. Нажмите на кнопку  Добавить новый объект на панели инструментов.
  4. На экране появится окно со списком платежных поручений данного плательщика, по которым имеется остаток:

 

 

  1. Выберите в списке платежное поручение, по которому следует осуществить перезачет и нажмите на кнопку Выбрать.
  2. На экране появится окно с данными по платежному поручению, с которого производится перезачет.

 

 

  1. В поле Сумма возврата укажите сумму, которая будет перезачтена плательщику.
  2. Нажмите на кнопку ОК.
  3. Платежное поручение будет добавлено в список Связанных документов.
  4. Можно добавить еще один платежный документ, повторив действия 4-9.
  5. При вводе сумм для перезачета общая сумма перезачета заполняется автоматически.
  6. По окончании ввода всех сумм закройте окно.

 

 

Чтобы найти документ на перезачет:

  1. Нажмите на кнопку  на панели инструментов Списка возвратов.

На экране появится панель для задания параметров поиска Поиск объектов.

  1. Задайте критерии поиска, заполнив следующие поля:
    • Номер;
    • Дата;
    • Плательщик;
    • ИНН;
    • КПП.

 

Примечание

Задание условия производится посредством ввода в поле значения, которому оно должно соответствовать. Для всех полей, кроме дат, можно вводить неполное значение (только начальные символы). Если значения указаны для нескольких полей, то система будет выбирать объекты, удовлетворяющие всем условиям одновременно. В условии можно использовать символы подстановки: “%” – любое количество любых символов, “_” – один произвольный символ.

Например, чтобы найти все объекты, содержащие часть слова пошлина, в параметрах поиска следует указать пошлин% или %ошлин%

Чтобы найти объект, содержащий в номере символ «4», в параметрах поиска следует указать %4% (будут выведены все объекты, содержащие символ «4» в середине номера) или 4% (будут выведены все объекты, содержащие в начале символ «4»).

 

  1. Нажмите на кнопку Искать.

На экран будут выведены результаты поиска, осуществленного в соответствии с заданным параметром.

 

Чтобы просмотреть данные документа на перезачет:

  1. Выберите в списке нужный документ.
  2. Нажмите на кнопку  Открыть выбранные файлы на панели инструментов окна Перезачеты.

На экране появится детальная форма Документа на перезачет.

 

 

Чтобы утвердить документ на перезачет:

  1. Выберите в списке нужный документ.
  2. Нажмите на кнопку  Открыть выбранные файлы на панели инструментов окна Перезачеты.

На экране появится детальная форма Документа на перезачет.

  1. Нажмите на кнопку Формируется на панели инструментов и выберите из списка пункт Утвердить.
  2. На экране появится окно для подтверждения:

 

 

  1. Нажмите Да, если данный платеж действительно требуется утвердить.

 

 

Чтобы удалить документ на перезачет:

  1. Выберите в списке документ на перезачет, который необходимо удалить.
  2. Нажмите на кнопку  Удалить выделенные объекты на панели инструментов списка перезачетов.

На экране появится запрос на подтверждение удаления выбранного документа.

  1. Нажмите Да, если данный документ на перезачет действительно требуется удалить.

Выбранный документ будет удален из списка перезачетов.

После удаления эти суммы вернутся на исходные платежные документы, с которых формировался перезачет.
