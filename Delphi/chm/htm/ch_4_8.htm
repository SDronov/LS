4.8.       Платежи для перечисления в бюджет

Данные по платежам для перечисления в бюджет доступны в меню панели навигации Документы -> Платежи в бюджет.

 

 

Документы выделяются в списке различными цветами:

Красным – подготавливаемый документ (не отправленный).

 

Для выполнения операций с платежами для перечисления в бюджет воспользуйтесь кнопками на панели инструментов.

 

Чтобы создать новый платеж для перечисления в бюджет:

  1. Нажмите на кнопку  Добавить новый объект на панели инструментов Платежей в бюджет.

На экране появится окно с доступными суммами для перечисления в бюджет:

 

 

 

Примечание

Суммы для перечисления в бюджет формируются программой автоматически на основании списаний с платежных документов.

В суммы для перечисления в бюджет не включаются следующие суммы:

-       суммы списаний по платежным документам «на контроле» до момента поступления этих средств на счет таможни;

-       суммы списаний по таможенной расписке.

 

 

  1. Выберите нужную сумму и нажмите на кнопку ОК.
  2. В списке появится новый платеж для перечисления в бюджет или выбранная сумма будет добавлена в подготавливаемый платеж для перечисления в бюджет, если он был создан ранее.

 

Чтобы найти платеж для перечисления в бюджет:

1.     Нажмите на кнопку  на панели инструментов Платежей в бюджет.

На экране появится панель для задания параметров поиска Поиск объектов.

2.     Задайте критерии поиска, заполнив поля Номер, Дата с, Дата по, Валюта.

 

Примечание

Задание условия производится посредством ввода в поле значения, которому оно должно соответствовать. Для всех полей, кроме дат, можно вводить неполное значение (только начальные символы). Если значения указаны для нескольких полей, то система будет выбирать объекты, удовлетворяющие всем условиям одновременно. В условии можно использовать символы подстановки: “%” – любое количество любых символов, “_” – один произвольный символ.

Например, чтобы найти все объекты, содержащие часть слова пошлина, в параметрах поиска следует указать пошлин% или %ошлин%

Чтобы найти объект, содержащий в номере символ «4», в параметрах поиска следует указать %4% (будут выведены все объекты, содержащие символ «4» в середине номера) или 4% (будут выведены все объекты, содержащие в начале символ «4»).

 

3.     Нажмите на кнопку Искать.

На экран будут выведены результаты поиска, осуществленного в соответствии с заданными параметрами.

 

 

Чтобы просмотреть данные по платежу для перечисления в бюджет:

1.     Выберите в списке нужный платеж для перечисления в бюджет.

2.     Нажмите на кнопку  Открыть выбранные файлы на панели инструментов окна Платежи в бюджет.

На экране появится детальная форма Платежа в бюджет.

 

На закладке Платеж в бюджет  представлены основные данные по платежу для перечисления в бюджет.

 

 

На закладке Платежи представлен список всех списаний, по которым осуществляется платеж для перечисления в бюджет.

 

 

 

 

Чтобы отправить платеж для перечисления в бюджет:

1.     Откройте детальную форму Платежа в бюджет и нажмите на кнопку Отправить на панели инструментов на закладке Платеж в бюджет.

На экране появится окно для ввода данных:

 

 

2.     Укажите в окне номер платежа для перечисления в бюджет и дату отправки.

3.     Нажмите на кнопку ОК.

 

 

Чтобы отменить отправку платежа для перечисления в бюджет:

1.     Откройте детальную форму Платежа в бюджет и нажмите на кнопку Отмена на панели инструментов на закладке Платеж в бюджет.

На экране появится окно для подтверждения:

 

 

2.     Нажмите Да, если данный платеж действительно требуется отменить.

 

 

Чтобы удалить платеж для перечисления в бюджет:

  1. Выберите в списке платеж, который необходимо удалить.
  2. Нажмите на кнопку  Удалить выделенные объекты на панели инструментов списка платежей для перечисления в бюджет.

На экране появится запрос на подтверждение удаления выбранного платежа.

  1. Нажмите Да, если данный платеж действительно требуется удалить.

Выбранный платеж будет удален из списка платежей для перечисления в бюджет.

 

Примечание

Для удаления доступны только подготавливаемые платежи для перечисления в бюджет. Если платеж уже отправлен, но его всё-таки необходимо удалить, то сначала надо отменить отправку, а потом уже удалить.

После удаления эти суммы станут доступны для формирования нового платежа для перечисления в бюджет.
