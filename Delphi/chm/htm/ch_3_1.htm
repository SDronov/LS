3.1.       Настройка Excel

Для формирования отчетов на компьютере должно быть установлено приложение Excel версии 2000 или выше.

Для корректной работы с отчётами необходимо настроить приложение Excel на каждом рабочем месте, где будет проходить формирование отчётов.

Для настройки Excel

  1. Запустите приложение Excel.
  2. Выполните команду Параметры из пункта меню Сервис. На экране появится окно Параметры.

Окно Параметры

  1. Перейдите на закладку Безопасность.
  2. Нажмите на кнопку Безопасность макросов…. На экране появится окно Безопасность.
  3. Перейдите на закладку Надёжные источники.

Окно Безопасность

  1. Установите флажок Доверять доступ к Visual Basic Project.
  2. Нажмите в окне Безопасность кнопку ОК.
  3. Нажмите в окне Параметры кнопку ОК.
  4. Закройте приложение Excel.
