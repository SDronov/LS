4.9.       Временный ввоз

Данный режим предназначен для контроля за уплатой периодических платежей по товарам, находящимся в режиме временного ввоза.

В данный режим автоматически попадают ГТД с режимом ИМ31.

 

Данные по ГТД с режимом временного ввоза доступны в меню панели навигации Контроль -> Временный ввоз.

 

Список ГТД в режиме временного ввоза

Документы выделяются в списке различными цветами:

Голубым – ГТД с ошибками;

Красным – ГТД с задолженностью;

Зеленым – ГТД крупных плательщиков.

 

В списке документов предусмотрены следующие виды фильтрации:

Показать:

  • Все – в списке отображаются все документы
  • Только хорошие – в списке отображаются только ГТД без ошибок,
  • Только с ошибками - в списке отображаются только ГТД с ошибками.

Код: - для отбора ГТД по указанному посту.

 

Чтобы найти ГТД:

1.     Нажмите на кнопку  на панели инструментов Списка ГТД.

На экране появится панель для задания параметров поиска Поиск объектов.

2.     Задайте критерии поиска, заполнив следующие поля:

    • Код таможни (поста)
    • Дата ГТД (период);
    • Номер;
    • ИНН (декларанта);
    • КПП (декларанта);

 

Примечание

Задание условия производится посредством ввода в поле значения, которому оно должно соответствовать. Для всех полей, кроме дат, можно вводить неполное значение (только начальные символы). Если значения указаны для нескольких полей, то система будет выбирать объекты, удовлетворяющие всем условиям одновременно. В условии можно использовать символы подстановки: “%” – любое количество любых символов, “_” – один произвольный символ.

Например, чтобы найти все объекты, содержащие часть слова пошлина, в параметрах поиска следует указать пошлин% или %ошлин%

Чтобы найти объект, содержащий в номере символ «4», в параметрах поиска следует указать %4% (будут выведены все объекты, содержащие символ «4» в середине номера) или 4% (будут выведены все объекты, содержащие в начале символ «4»).

 

3.     Нажмите на кнопку Искать.

На экран будут выведены результаты поиска, осуществленного в соответствии с заданными параметрами.

 

 

Чтобы просмотреть данные по ГТД:

1.     Выберите в списке нужную ГТД.

2.     Нажмите на кнопку  Открыть выбранные файлы на панели инструментов окна Список ГТД.

На экране появится детальная форма ГТД.

Помимо стандартных закладок добавляется закладка Импорт 31.

 

 

Чтобы ввести оплату периодических платежей:

  1. Нажмите на кнопку Оплатить на панели инструментов.
  2. На экране появится окно Выбор месяца для оплаты.

Выберите месяц, по который включительно и нажмите на кнопку ОК.

Тем самым будет определена сумма для уплаты.

 

  1. Появится Мастер оплаты.

 

 

  1. Нажмите на кнопку Добавить.
  2. Появится Список платежных документов. Найдите нужный документ и нажмите на кнопку Выбрать. В появившемся окне введите сумму списания с документа и нажмите ОК.

 

 

  1. Если сумма списания меньше суммы корректировки, то в мастере оплаты можно добавить еще один платежный документ или поставить галочку Перевести остаток в задолженность.
  2. Нажмите на кнопку Готово.

 

Чтобы рассчитать и начислить проценты при выпуске в свободное обращение:

  1. Нажмите на кнопку Смена режима на панели инструментов.
  2. На экране появится окно Выбор товаров и количества.

Введите нужное количество товаров для выпуска в свободное обращение и нажмите на кнопку ОК.

 

Примечание

При расчете процентов используется справочник Ставки ЦБ. Перед начислением процентов убедитесь, что данный справочник содержит правильные значения.
