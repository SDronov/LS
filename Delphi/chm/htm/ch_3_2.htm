3.2.       Операции с НСИ

Для работы с КПС "Лицевые счета" используется нормативно-справочная информация. Все справочники НСИ должны быть заполнены. При инсталляции системы справочник Коды бюджетной классификации поставляется заполненным. Остальные справочники необходимо импортировать.

 

Чтобы импортировать справочники:

1.     На главной панели инструментов выберите пункт Ввод НСИ;

2.     Выберите один из справочников, например Участники ВЭД.

3.     Укажите путь к файлу, из которого импортируется справочник:

 

Импорт справочника

4.     Нажмите на кнопку Открыть.

На экране появится сообщение следующего вида:

 

5.     Нажмите на кнопку ОК.

 

Аналогичные действия по импорту справочников необходимо провести с остальными справочниками НСИ.

 

 

Чтобы осуществить поиск по справочнику:

1.     Откройте нужный справочник, выбрав соответствующий пункт на панели навигации.

2.     Нажмите на кнопку  на панели инструментов окна справочника.

В окне справочника откроется панель для задания параметров поиска.

3.     Введите название или часть названия значения из справочника, которое следует найти.

 

 

Примечание

Задание условия производится посредством ввода в поле значения, которому оно должно соответствовать. Для всех полей, кроме дат, можно вводить неполное значение. Если значения указаны для нескольких полей, то система будет выбирать объекты, удовлетворяющие всем условиям одновременно. В условии можно использовать символы подстановки: “%” – любое количество любых символов, “_” – один произвольный символ.

 

 

4.     Нажмите на кнопку Искать.

Результаты поиска по справочнику Виды платежей

Для эффективной работы со справочниками предусмотрена панель инструментов. Поместив курсор мыши над кнопкой панели, можно увидеть ее название, которое является кратким описанием функции этой кнопки.

 

 

Чтобы просмотреть данные значения из справочника:

  1. Выберите в списке нужное значение справочника.
  2. Нажмите на кнопку  Открыть выбранные файлы на панели инструментов окна справочника.

На экране появится детальная форма Значения справочника.

 

 

Чтобы создать новое значение в справочнике:

  1. Нажмите на кнопку  Добавить новый объект на панели инструментов окна справочника.

На экране появится детальная форма Ввода параметров нового значения справочника.

  1. Заполните необходимые поля и нажмите на кнопку  Сохранить.

Новое значение будет добавлено в справочник.

 

 

Чтобы удалить значение из справочника:

  1. Выберите в списке значение справочника, которое необходимо удалить.
  2. Нажмите на кнопку  Удалить выделенные объекты на панели инструментов окна справочника.

На экране появится запрос на подтверждение удаления выбранного значения из справочника.

  1. Нажмите Да, если данное значение справочника действительно требуется удалить.

Выбранное значение будет удалено из справочника.
