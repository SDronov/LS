3.5.5.  Перенос данных из буфера в основную базу

Чтобы перенести данные из буфера в основную базу

1.     Нажмите на кнопку Перенести на панели меню рабочей области.

2.     В появившемся окне введите дату, на которую переносятся остатки.

По умолчанию предлагается 31.12.2005. При необходимости её можно изменить.

 

 

Нажмите кнопку Далее.

3.     Далее будет происходить обработка данных, о чем выдастся соответствующее окно

 

 

4.     По окончании обработки будет выведен протокол.

 

 

5.     Нажмите на кнопку Готово.

 

Если при переносе данных возникли новые ошибки, которых не было перед переносом, то такие записи можно исправить и повторить перенос. Ранее перенесенные данные повторно не переносятся, перенесутся только новые и исправленные записи.
