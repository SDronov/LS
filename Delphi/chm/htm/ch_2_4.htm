2.4.       Панель инструментов

Окно, содержащее список объектов, содержит панель инструментов, предназначенную для управления приложением. Кнопки этой панели описаны в следующей таблице.

 

Кнопка

Команда

Описание

Добавить новый объект

Добавить новый объект в список объектов.

Открыть выбранные объекты

Открыть выбранные объекты в новом окне.

Обновить представление

Обновить информацию на экране из базы данных.

Показать/скрыть панель поиска

Вывести/убрать панель поиска на экране.

Автофильтр данных таблицы

Вывести поля для фильтрации данных на экран.

Удалить выделенные объекты

Удалить выделенные объекты из базы данных.

Загрузка данных из справочника

Загрузить данные из выбранного справочника в систему.
