3.5.2.  Загрузка данных из внешних источников

Чтобы загрузить данные по остаткам:

1.     Нажмите на кнопку Загрузить… на панели меню рабочей области.

В появившемся окне выберите источник данных для загрузки:

 

Выбор источника данных для загрузки

 

ДОХОД 2.11 (2.12) - загрузить данные по остаткам из базы данных КПС «Лицевые счета» ИРС «Доход» версии 2.11 или 2.12;

Текстовые файлы - загрузить данные из текстовых файлов специального формата (формат текстовых файлов и пример их формирования описаны в документе Описание переноса остатков из текстовых файлов).

 

Нажмите на кнопку Далее.

 

2.     Укажите путь к файлам

 

 

Для загрузки данных из КПС «Лицевые счета» ИРС «Доход» версии 2.11 укажите путь к папке BASES в каталоге установки серверной части программы (по умолчанию LS_DB).

Для загрузки данных из текстовых файлов укажите путь к каталогу, в котором находятся эти файлы.

 

3.     Нажмите на кнопку Далее.

На экране появится окно с информацией о результате загрузки:

 

 

4.     Нажмите на кнопку Готово.

Загруженные данные помещаются в буфер.
