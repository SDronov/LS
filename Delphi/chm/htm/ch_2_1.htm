2.1         Запуск КПС "Лицевые счета"

Чтобы запустить КПС «Лицевые счета»:

1.     Воспользуйтесь ярлыком запуска КПС "Лицевые счета", созданном при установке программы.

На экране появится диалоговое окно входа в систему.

 

Диалоговое окно входа в систему

 

2.     Для входа в систему введите имя пользователя, пароль и имя базы данных, которые должен сообщить администратор.

3.     Нажмите на кнопку ОК.

 

В случае успешной аутентификации пользователя можно приступить к работе с КПС "Лицевые счета". Если при вводе данных произошла ошибка, например, неправильно введен пароль, система выдаст информацию об ошибке и попросит повторно заполнить необходимые поля.

 

Программа запоминает последние успешно введенные данные и при следующем входе программы достаточно будет ввести только пароль.

 

 

Если необходимо изменить данные для входа, нажмите Сменить пользователя.

 

Взаимодействие пользователя с КПС «Лицевые счета» происходит с помощью следующих элементов:

·       Рабочая область

·       Меню

·       Панель инструментов
