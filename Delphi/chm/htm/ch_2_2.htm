2.2.       Рабочая область

Рабочая область КПС «Лицевые счета» состоит из следующих элементов интерфейса:

Закладки детальной формы объекта
 

Окно списка объектов
 

Навигатор
 

Панель навигации
 

Главное меню
 

Детальная форма объекта
 

Панель инструментов
 

Панель для задания параметров поиска
 

Закладка Буфера объектов
 
 

 


Главное меню КПС «Лицевые счета» описаны в подразделе Главное меню ниже.

 

Панель навигации состоит из пунктов меню, по которым осуществляется вызов окна со списком объектов. Панель навигации разделена на предметные области, содержащие в себе пункты меню.

 

Навигатор. В окне Навигатора отображаются все окна, открытые пользователем в системе. Переход между окнами осуществляется при нажатии на название окна кнопкой мыши.

 

Буфер объектов. В окне Буфера объектов отображаются все объекты, помещенные в буфер. Работа с буфером объектов осуществляется через Главное меню или функциональными клавишами.

 

Панель инструментов описана в подразделе 2.3. Панель инструментов ниже.

 

Окно списка объектов. В окне списка объектов отображаются все объекты из базы данных системы, в зависимости от того, какой список выводится в данном окне.

 

Панель для задания параметров поиска. Панель предназначена для ввода параметров поиска в окне списка объектов.

 

Детальная форма объекта предназначена для детального отображения свойств объекта.

 

Закладки детальной формы объекта обеспечивают переход между различными данными выбранного объекта.
