Remarks
Является базовой для других функций/процедур работы с окнами сообщений.

Кнопка mbHelp не может быть использована.

procedure TfdcMessageForm.HelpButtonClick(Sender: TObject); begin if HelpContext = 0 then ShowHelpContents('') else ShowHelpContent('', HelpContext); end;
