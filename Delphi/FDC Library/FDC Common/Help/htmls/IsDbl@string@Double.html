IsDbl
fdcUtils | Example
function IsDbl(S: string; var AResult: Double): Boolean;
File
fdcUtils
Returns
Возвращет True, если строка S является числом с плавающей точкой, иначе возвращает False.
Description
Определение, является ли строка S числом с плавающей точкой.
Remarks
Если строка S является числом с плавающей точкой, то присваивает полученное число переменной AResult.
Example
    var
      v: double;
    begin
      if IsDbl(s, v) then
        Cell.SetValue(v);
    end;
