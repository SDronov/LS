procedure BeginBusy;
File
fdcUtils
Description
Устанавливает курсор мыши в приложении в crHourGlass.
Remarks
Предназначена для управления курсором во время длительных операций и предотвращает эффект "мигания" при частой смене курсора.
Notes
Каждому вызову BeginBusy должен соответствовать вызов EndBusy.
Example
    BeginBusy;
    try
      ...
    finally
      EndBusy;
    end;
