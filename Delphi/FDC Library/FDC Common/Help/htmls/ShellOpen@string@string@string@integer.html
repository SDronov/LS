ShellOpen
fdcUtils
procedure ShellOpen(AFileName: string; AParameters: string = ''; ADirectory: string = ''; AShowCmd: integer = SW_SHOWNORMAL);
File
fdcUtils
Parameters
Parameters Description
AFileName: string Имя файла, папки, документа, URL ...
AParameters: string = '' Cтрока параметров запускаемой программы
ADirectory: string = '' Директория по умолчанию
AShowCmd: integer = SW_SHOWNORMAL Состояние окна запущенной программы
Description
Открытие файла средствами Windows Shell. Запускает оболочку, ассоциированную с расширением файла, передаваемого ей в качестве параметра.
Remarks
Если AFileName - приложение, то AParameters может содержать строку параметров приложения. Если AFileName - документ или папка, AParameters должен быть nil.
