procedure GetFileVersion(AFileName: PChar; const AWhat: array of string; AVerInfo: array of PString);
File
fdcUtils
Parameters
Parameters Description
AFileName: PChar Имя файла, информацию о версии которого необходимо получить
const AWhat: array of string Открытый (динамически формирующийся в момент вызова) массив строк-запросов по разделам версии. Допустимые строки-запросы:

'CompanyName', 'FileDescription', 'FileVersion', 'InternalName', 'LegalCopyright', 'OriginalFilename', 'ProductName', 'ProductVersion'.

AVerInfo: array of PString Cтрока параметров запускаемой программы
Description
Определение информации о версии исполняемого файла.
Remarks
Сопоставление элементов AWhat и AVerInfo - позиционное: n-ый элемент aVerInfo - ответ на запрос в n-ом элементе AWhat. Если размеры в AWhat и aVerInfo различны, aVerInfo содержит ответы на первые n запросов из AWhat, где n = Min(размер AVerInfo, размер AWhat), при n < размер AVerInfo, оставшиеся элементы массива инициализируются пустой строкой.
Example
  var
    InternalName, FileVersion: string;
  begin
    GetFileVersion(PChar(Application.ExeName),
      ['InternalName', 'FileVersion'],
      [@InternalName, @FileVersion]);
    Label1.Caption := InternalName + ' ' + FileVersion;
  end;
