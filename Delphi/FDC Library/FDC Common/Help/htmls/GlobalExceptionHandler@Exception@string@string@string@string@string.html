procedure GlobalExceptionHandler(AException: Exception; AUserMessage: string; AUserHint: string; AUnitName: string; AClassName: string; AProcName: string);
File
fdcExceptions
Parameters
Parameters Description
AException: Exception Исключительная ситуация для обработки
AUserMessage: string Пользовательское сообщение
AUserHint: string Подсказка пользователю
AUnitName: string Имя модуля, в котором возникла исключительная ситуация
AClassName: string Имя класса, в котором возникла исключительная ситуация
AProcName: string Имя процедуры, в которой возникла исключительная ситуация
Description
Метод обработки исключительной ситуации.
Remarks
Обработав исключительную ситуацию, метод GlobalExceptionHandler создаст новую исключительную ситуацию типа EfdcException. Таким образом, исходная исключительная ситуация будет потеряна.
Notes
При незаполненных параметрах AUserMessage и AUserHint соответствующие поля EfdcException будут инициализированы значениями SDefaultUserMessage и SDefaultUserHint соответственно;
Example
  unit UnitA;
  ...
  procedure TClassA.MethodA;
  begin
    try
      ...
    except
      on E: Exception do
        GlobalExceptionHandler(
          E,
          'Пользовательское сообщение',
          'Подсказка пользователю',
          'UnitA',
          ClassName,
          'MethodA');
    end;
  end;
  ...
  end.
