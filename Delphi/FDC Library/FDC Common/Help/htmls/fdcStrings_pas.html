Function Description
DeleteSubString Возвращает строку полученную удалением ACount символов из строки ASourceStr, начиная с позиции AIndex
GetWord Возвращает слово с номером AWordNo в строке, разделенной символом ASeparator
GetWordCount Возвращает количество слов в строке AString разделенной символом ASeparator
IsBeginWith Проверяет начинается ли строка ASourceStr подстрокой AWithStr
IsEndWith Проверяет заканчивается ли строка ASourceStr подстрокой AWithStr
IsMaskedString Проверяет соответствует ли строка AStr маске AMask.
